# Public-release checklist

- [ ] All coauthors approved public release.
- [ ] Repository name and URL are final.
- [ ] `CITATION.cff` contains the final title, URL, DOI, ORCIDs, and version.
- [ ] Raw-data ownership and reuse permissions are documented.
- [ ] Personal, confidential, or proprietary metadata were removed.
- [ ] The manuscript version is legally permitted for redistribution.
- [ ] Every figure can be regenerated by code or is documented as external.
- [ ] A clean environment reproduces the reported coefficients and metrics.
- [ ] Tests pass.
- [ ] A GitHub release `v1.0.0` was created.
- [ ] The release was archived and assigned a DOI.
- [ ] The repository DOI was added to the manuscript/data-availability statement.
